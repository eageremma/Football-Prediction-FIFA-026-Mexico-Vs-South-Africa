# Data Sources

## Historical Results

The historical results dataset contains international football matches.

Expected columns:

- date
- home_team
- away_team
- home_score
- away_score
- tournament
- city
- country
- neutral

Suggested free source:

- Kaggle: International football results from 1872 onwards

## Elo Ratings

Elo ratings are used as a measure of team strength.

For this project:

| Team | Elo |
|---|---:|
| Mexico | 1875 |
| South Africa | 1517 |

Suggested free source:

- World Football Elo Ratings

## Match Context

The match context dataset includes:

- host nation advantage
- altitude factor
- opening match pressure
- home crowd advantage
- South Africa early arrival
- South Africa preparedness
- tactical context
- source URL

Suggested sources:

- FIFA
- Reuters
- ESPN
- Guardian

## Why Multiple Sources Are Used

One dataset alone cannot explain a football match properly. Historical data tells us what happened in the past. Elo tells us team strength. Match context tells us what is special about this fixture.
