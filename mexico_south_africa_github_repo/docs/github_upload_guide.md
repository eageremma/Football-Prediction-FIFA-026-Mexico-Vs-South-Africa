# GitHub Upload Guide

## Option 1: Upload through GitHub website

1. Go to GitHub.
2. Click the + icon.
3. Select New repository.
4. Repository name suggestion:
   `mexico-south-africa-worldcup-prediction`
5. Add this description:
   `Sports analytics project predicting Mexico vs South Africa using Python, Elo ratings, Poisson modelling, and machine learning.`
6. Keep it Public if you want recruiters to see it.
7. Create repository.
8. Click Add file > Upload files.
9. Drag and drop all project files and folders.
10. Click Commit changes.

## Option 2: Upload with Git command line

```bash
git init
git add .
git commit -m "Initial commit: Mexico vs South Africa football prediction analysis"
git branch -M main
git remote add origin https://github.com/your-username/mexico-south-africa-worldcup-prediction.git
git push -u origin main
```
